Defines the primitives from which all InfoGrid models and instances are built.

This includes, most importantly:

  • {@link org.infogrid.model.primitives.EntityType EntityType}: the type for objects that can stand alone, i.e. that do not depend on other objects (i.e. {@link org.infogrid.mesh.MeshObject MeshObjects})
  • {@link org.infogrid.model.primitives.RelationshipType RelationshipType}: the type for objects that depends on two other objects (i.e. relationships between pairs of MeshObjects.
  • {@link org.infogrid.model.primitives.PropertyType PropertyTypes}: types of properties of MeshObjects. PropertyTypes have {@link org.infogrid.model.primitives.DataType DataTypes}, they can be mandatory or optional etc.
  • {@link org.infogrid.model.primitives.SubjectArea SubjectArea}: a packaging mechanism for InfoGrid models, similar to the Java package concept
  • {@link org.infogrid.model.primitives.PropertyValue PropertyValue}: with many subtypes for the particular value types that PropertyTypes can have.

To instantiate a model, use a (subtype of) {@link org.infogrid.modelbase.ModelBase ModelBase}.
