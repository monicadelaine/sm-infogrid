Tag libraries and utility classes for custom JSP tags that are InfoGrid-aware. Sub-packages contain specific tag libraries.

The following tag libraries are available:

org.infogrid.jee.taglib.candy
Eye candy.
org.infogrid.jee.taglib.logic
Tags evaluating logic conditions, such as conditionals based on PropertyValues
org.infogrid.jee.taglib.mesh
Tags that render MeshObjects and their characteristics in a variety of ways, such as showing the time the MeshObject was last updated, or the set of roles in which it currently participates.
org.infogrid.jee.taglib.mesh.set
Tags that render MeshObjectSets (i.e. sets of MeshObjects)
org.infogrid.jee.taglib.meshbase
Tags that deal with MeshBases rather than MeshObjects
org.infogrid.jee.taglib.util
Utility tags
org.infogrid.jee.taglib.viewlet
Tags that make Viewlets easier to create
org.infogrid.jee.taglib.templates
Tags that interact with the templating system

All InfoGrid tag libraries following these conventions for the attributes on the tag in the HTML:

Attribute name Meaning
xxxName Name of a Bean in the page (or other) context, where xxx indicates the semantics of the Bean.
name Name of a Bean that is the primary subject of the tag.
xxxLoopVar Name of the Bean created as the loop variable in a tag that may iterate over its body, where xxx indicates the semantics of the Bean.
