A templating mechanism for InfoGrid JEE applications.

All output is deposited in an instance of class {@link org.infogrid.jee.templates.StructuredResponse StructuredResponse}, which is composed of one or more named {@link org.infogrid.jee.templates.StructuredResponseSection StructuredResponseSections}.

{@link org.infogrid.jee.templates.StructuredResponseTemplate StructuredResponseTemplate} is then used to assemble some or all of the StructuredResponseSections into the output stream.
